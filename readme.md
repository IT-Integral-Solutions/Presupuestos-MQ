# ✨ Presupuestos MQ

Sistema de generación y visualización de presupuestos HTML para **Magic Queens**, desarrollado por **IT Integral Solutions**.

Este repositorio contiene las plantillas y lógica necesarias para renderizar propuestas de viaje de forma clara, visual y profesional, optimizadas tanto para envío digital como para conversión comercial.

---

## 🚀 Objetivo

Brindar a las agencias una herramienta moderna para:

- Generar presupuestos visualmente atractivos
- Mejorar la tasa de conversión de clientes
- Estandarizar la presentación de propuestas
- Facilitar la personalización por marca

---

## 🧩 Características principales

- 📄 Plantillas HTML optimizadas para presupuestos de viajes
- 🎨 Diseño adaptable a branding de la agencia (colores, logos, estilos)
- 📱 Responsive (mobile-first)
- 🧾 Preparado para exportación a PDF
- 🔗 Integrable con CRM (TASS CRM u otros sistemas)
- ⚡ Render rápido y liviano
